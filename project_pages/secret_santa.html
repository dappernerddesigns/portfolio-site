Secret Santa

A React app built with Firebase

Using Firebase for a serverless back end this app stores user profiles, Secret Santa groups and allows the creation of new users and new groups. With a randomly generated invite code, group owners can share this code with their friends to join private groups. The group admin can set a closing date and randomly draw pairs for gifts.

Designed mobile first, this app is responsive and works across all browser sizes. Built with React, Tailwind and some free Christmassy assets, I hope to have this hosted and live for the next Christmas season!
